# Excel HR Analytics 
HR Analytics Dataset Created by Dr. Rich Huebner found on Kaggle: https://www.kaggle.com/datasets/rhuebner/human-resources-data-set?resource=download. 

## Data Overview
  This data was created by Dr. Carla Patalano and Dr. Rich Huebner as away to teach HR students how to use and analyze data. It's primary use is a teaching dataset--to teach
  human resources professionals how to work with data and analytics.

  The content of the data is of a single data file (v14). It consists of a fictitious company name and core data set icluding names, DOB's, age, gender, reasons for termination, 
  departments, performance scores, etc. You can read more and learn about this dataset and its creators at the link pasted above.

## Business Problem
  This analysis explores workforce trends at a mid-size fictional company to identify the main drivers of employee attrition.

  - Which departments have the highest attrition rate?
  - For what reasons do most employees leave the company?
  - Which managers have the highest attrition based on the size of their team?

## Key Insights
  - The production department drives nearly 80% of overall company attrition.
  - Two managers, Amy Dunn and Kissy Sullivan have the highest team attrition rates at 62% and 55% respectively. Recommend Review for these managers.
